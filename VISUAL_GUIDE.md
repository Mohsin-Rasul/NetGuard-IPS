## Data Display Formats

### Time Format
```

```

### IP Address Format
```
IPv4: 192.168.1.50
IPv6: 2001:db8::1
CIDR: 192.168.0.0/24

### File Sizes
```
Bytes:     1024 B

Megabytes: 2.5 MB
Gigabytes: 1.0 GB
```

### Packet Details
```
Protocol: TCP, UDP, ICMP, Other
Port:     0-65535
